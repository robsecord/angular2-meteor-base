# Angular2-Meteor-Base
Angular2 + Meteor -- Base Application

*Note: This is a Work-In-Progress*

## Current Versions
* Meteor: 1.3.5.1
* Angular: 2.0.0-rc5
* Twitter Bootstrap: 3.3.7

## Build & Run
1. git clone [this_repo]
2. cd angular2-meteor-base/
3. meteor npm install
4. meteor
