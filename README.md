# Angular2-Meteor-Base
Angular2 + Meteor -- Base Application

*Note: This is a Work-In-Progress*

## Current Versions
* Meteor: 1.3.5.1
* Angular: 2.0.0-rc5
* Twitter Bootstrap: 3.3.7

## Build & Run
* git clone [this_repo]
* cd angular2-meteor-base/
* meteor update --release 1.3.5.1
* meteor npm install
* meteor
